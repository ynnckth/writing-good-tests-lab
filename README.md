Run all tests of a specific module directory with:
> `./gradlew -p <dir> test`

###Topics Covered

- JUnit 5 (Jupiter)
   - Hamcrest Matchers
   - Extensions and Rules
   - Testdata Management (Parameterized tests)
- Test Design (design for testability)
- Code Coverage
- Test Doubles (Stubs, Mocks, Spies)
- Mockito
- Test driven development (TDD)
